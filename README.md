Movement - W A S D <br/>
Run - Shift <br/>
Dive - Right Button[Mouse] <br/>
Jump - Space <br/>
PickAndDrop - G <br/>
Crouch - Q <br/>
Spin - Left Button[Mouse] <br/>
Stomp - E <br/>
AirDive - Q <br/>
ReleaseLedge - Right Button <br/>
Glide - Right Button <br/>
Dash - F <br/>
Grind Brake - Q <br/>
Pause - Esc <br/>
